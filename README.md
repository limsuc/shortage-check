# shortage-check

거래처품절확인: 거래처 마스터와 품절 공지를 대조해 영향이 있는 거래처를 빠르게 확인하는 새 프로젝트입니다.

## 로컬 미리보기

정적 파일만으로 동작하는 프리뷰입니다.

```bash
python3 -m http.server 8770
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:8770
```

## 현재 범위

- MEDMATE 디자인 톤을 따르는 독립 화면
- CSV 거래처 마스터 업로드 및 파싱
- 서원파마 제공 품절목록 기본 불러오기
- PDF, CSV, TXT 품절목록 업로드 및 파싱
- 제품명/용량 기준 매칭
- 영향 거래처 결과 테이블
- 요약 복사 / 품절목록 엑셀 다운로드
- 거래처별 TEXT 메세지 복사
- 거래처별 품절내용 이미지 복사 또는 PNG 저장
